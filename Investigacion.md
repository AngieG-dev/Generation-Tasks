# ¿Qué es un sistema operativo?

Un **sistema operativo** es el software principal que administra el hardware y los recursos de un dispositivo, como un computador o teléfono. Funciona como un intermediario entre el usuario y la máquina, permitiendo que los programas se ejecuten correctamente.

## Funciones principales

- Gestiona el hardware (CPU, memoria y dispositivos).
- Permite ejecutar programas y aplicaciones.
- Administra archivos y almacenamiento.
- Facilita la conexión a redes y dispositivos.

**Importancia:**  
Sin un sistema operativo, el computador no podría iniciar ni ejecutar programas, ya que faltaría el software encargado de controlar y coordinar todos los componentes del sistema.

# ¿Qué es una terminal?

Una **terminal** es un medio donde un usuario puede "hablar" de forma directa con un computador, dándole instrucciones escritas en texto.

## ¿Para qué sirve?

Una terminal sirve para realizar tareas mucho más rápido usando el teclado, en lugar de navegar por menús utilizando el mouse.

## ¿Por qué los desarrolladores la usan tanto?

Los desarrolladores la utilizan principalmente por tres razones:

- **Velocidad**
- **Control**
- **Automatización**

En términos simples, es el camino corto para darle órdenes directas a la computadora.

---

# ¿Por qué es importante saber todo esto para un desarrollador?

Es importante porque son **herramientas básicas para un programador**, que permiten trabajar de forma más eficaz y ahorrar tiempo en los proyectos.

- **Terminal:** agiliza la comunicación entre el programador y el computador.
- **README:** funciona como una forma de comunicación e instrucciones entre quien creó un proyecto y quien lo utiliza.

# ¿Es obligatorio tener un sistema operativo para programar?

Depende del proyecto, ya que el sistema operativo establece ciertas reglas para crear distintos programas. Por eso, es importante considerar el sistema operativo en el que se usará el programa.

## ¿Por qué?

Los sistemas operativos funcionan de maneras bastante distintas, de la misma forma que una casa en Chile es estructuralmente distinta a una casa en Estados Unidos.
